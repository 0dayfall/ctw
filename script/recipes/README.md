# ctw Recipes

Short, copy-pasteable recipes for common workflows. These are safe defaults and
are meant to be stitched into your own scripts.

## Contents

- `watch_jsonl.sh` – stream tweets as JSON lines for ingestion
- `watch_pretty.sh` – compact pretty JSON for terminal use
- `watch_save_log.sh` – append to a daily log file
- `reconnect_summary.sh` – runs watch and highlights reconnects/summary output
